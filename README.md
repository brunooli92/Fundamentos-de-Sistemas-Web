O presente repositório foi criado pelo discente Carlos Bruno Cabral de Oliveira, aluno do curso Tecnologia em Análise e Desenvolvimento de Sistemas da PUCRS, para o Escritório de Projetos da disciplina Fundamentos de Sistemas Web. 

Para obtenção de nota na referida disciplina, está sendo desenvolvido um sistema web para um pet shop fictício chamado Pet Shop Miau, utilizando HTML5 para estabelecer a arquitetura básica do front-end.

Para a Fase 1 do projeto, o principal objetivo foi desenvolver o esqueleto estrutural do website para o pet shop, utilizando a semântica léxica do HTML5 para a fundação arquitetural de acordo com os requisitos da Fase 1.

Na fase 1, o sistema contava com duas páginas: “Início (Produtos”) e “Nossos serviços”. Ambas as páginas tinham uma estrutura padrão (header, body, footer) e utilizavam hiperlinks para a navegação:

<p> •	Header: Título principal do pet shop e o menu de navegação <!-- <nav> --> para acesso rápido às páginas. </p>
<p> •	Body: Conteúdo central e exclusivo de cada página. </p>
<p> •	Footer: Informações de autoria, contato e links legais, utilizando a tag  <!-- <address> --> para formatação de contato. </p>

Para a Fase 1, o sistema foi desenvolvido intencionalmente sem customizações (CSS e Bootstrap) e sem funções em JavaScript, conforme instruções para a Fase 1. Tais desenvolvimentos ocorreram na Fase 2.

Na fase 2, foi implementado:

- CSS/Bootstrap (carrossel) e uso de JavaScript;
- Formulário de cadastro do cliente e do pet com os dados de identificação do cliente e do pet;
- Elementos variados no formulário e atributos, como: input, check box, number, e-mail, radio button, placeholder, campos obrigatórios, etc.;
- Escolha do serviço e método de agendamento, com definição do dia e horário de agendamento;
- Acessibilidade para pessoas com deficiência visual por meio do uso de tags semânticas, com todas as imagens possuindo o atributo alt para descrição de imagem.
